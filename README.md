Project Overview: Ticketify
Ticketify is a premium, full-stack Railway Reservation System built to provide a seamless booking experience with a state-of-the-art aesthetic.

✨ Visual & UI Highlights
Premium Design: Features a curated Gold & Obsidian Black color palette for a high-end, luxury feel.
Glassmorphism: Modern UI components with blur effects, subtle transparency, and refined borders.
Dynamic Animations: An interactive hero section with an animated train on glowing track lines and success screens with pop-up animations.
Intuitive UX: Responsive navigation, smart search fields, and a redBus-inspired dashboard.
🚀 Key Features
Smart Search & Booking: Effortlessly find trains, check seat availability, and book tickets in real-time.
User Dashboard: Personalized profile management, booking history, and account settings.
Admin Command Center: A comprehensive dashboard for managing trains, schedules, seat layouts, and user data.
Simulated Payments: Integrated payment processing flow for a complete end-to-end user journey.
🛠️ Technical Stack
Backend: Java 17+, Spring Boot 3.2.4, Spring Data JPA.
Database: H2 (Persistent File-based) for reliable local storage and MySQL compatibility.
Frontend: HTML5, Advanced CSS3 (Custom Design System), and Vanilla JavaScript.
Security: Spring Security integration for authentication and protected admin routes.
